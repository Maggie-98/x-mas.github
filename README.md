# 圣诞节礼物互换 - X-mas Gift Exchange

这是一个简单的圣诞节礼物互换，

This project is a simple Christmas Gift Exchange system, where users can randomly select a name for a gift exchange via the browser. The system supports both Chinese and English languages.

## 功能 / Features

1. **选择名字** / **Choose a Name**: 每个参与者从名单中选择一个名字，避免选择自己。  
   Each participant selects a name from the list, but cannot choose themselves.

2. **抽选牌** / **Draw a Card**: 每人只能抽选一次和抽选一个   
   Each person can only draw once, and cannot re-draw once they have selected.

3. **总结页面** / **Summary Page**: 展示每个抽选者的选项，记录下抽选者与被抽选者以及抽选的日期时间。  
   Displays the summary of each participant’s selection, including who they drew and the date/time of the selection.
