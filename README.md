# 圣诞节礼物互换 - X-Mas Gift Exchange 🎄🎁

这是一个简单的圣诞节礼物互换网页应用，帮助用户与朋友们一起交换圣诞礼物。通过该应用，用户可以选择一个名字，随机抽取一个礼物交换对象，使用翻牌模式来展示交换结果。

## 功能概述

该应用包含三个主要步骤：

1. **选择名字**  
   用户从预设的名字列表中选择一个名字进行礼物交换，确保选择的名字不是自己。

2. **翻牌模式**  
   用户点击卡片，系统会随机抽取一个礼物交换对象，且每个用户只可进行一次翻牌。

3. **总结页面**  
   显示用户抽选的结果，告诉用户他们为谁准备了礼物。

## 项目结构

该项目由以下文件组成：

- `index.html` - 页面内容结构，包含选择名字、翻牌抽选、显示结果部分。
- `style.css` - 页面样式文件，定义了页面的视觉效果和布局。
- `script.js` - JavaScript 文件，控制页面的交互和逻辑。

### `index.html`

该文件定义了页面的结构，包括：
- 一个标题（`🎄 圣诞节礼物互换 / X-Mas Gift Exchange 🎁`）
- 一个下拉列表，供用户选择名字。
- 一个按钮，用户点击后系统会随机抽选一个礼物交换对象（翻牌模式）。
- 显示抽选结果。

### `style.css`

该文件定义了页面的样式：
- 页面背景设置为圣诞节的红色，并使用白色字体。
- 按钮和文本的样式美化，提供了交互效果（如按钮悬停效果）。
- 翻牌区域使用卡片样式，用户点击后会显示交换结果。

### `script.js`

该文件包含了应用的逻辑：
- 用户选择名字后，点击按钮，系统会随机抽取一个名字，显示为礼物交换对象。
- 使用翻牌效果，用户点击卡片时显示抽选结果。
- 每个用户仅能进行一次抽选，且每次抽选会显示唯一的结果。
